SECTION 6: KEY PYTHON CONCEPTS COVERED IN THIS COURSE
================================================================


PYTHON_BASICS:
- What is Python: Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in cloud computing, data science, AI, and web development.
- Installing Python: Download from python.org. Verify installation by running "python --version" in terminal.
- Python Syntax: Python uses indentation (spaces or tabs) to define blocks of code. No curly braces needed.
- Comments: Use # for single-line comments. Use triple quotes for multi-line comments.
- print() function: Used to display output. Example: print("Hello, World!")
- input() function: Used to take user input. Example: name = input("Enter your name: ")


VARIABLES_AND_DATA_TYPES:
- Variables store data. No need to declare type. Example: x = 5, name = "Riwaj", is_active = True
- int: whole numbers. Example: age = 21
- float: decimal numbers. Example: price = 9.99
- str: text strings. Example: name = "Nepal"
- bool: True or False
- Casting: converting between types. int("5") converts string to int. str(5) converts int to string. float("3.14") converts string to float.
- type() function: returns the data type. Example: type(5) returns int.


STRINGS:
- Strings are sequences of characters enclosed in quotes.
- String methods: .upper(), .lower(), .strip(), .replace(), .split(), .len()
- String concatenation: "Hello" + " " + "World"
- f-strings: f"My name is {name}" for formatted strings
- String indexing: name[0] gives first character. name[-1] gives last character.
- String slicing: name[0:3] gives characters from index 0 to 2.


BOOLEANS:
- Boolean values are True or False.
- Comparison operators return booleans: ==, !=, >, <, >=, <=
- Logical operators: and, or, not


OPERATORS:
- Arithmetic: + - * / // % **
- Assignment: = += -= *= /=
- Comparison: == != > < >= <=
- Logical: and or not
- Floor division //: returns the integer part. 10 // 3 = 3
- Modulus %: returns the remainder. 10 % 3 = 1
- Exponentiation **: 2**3 = 8


FLOW_CONTROL:
- if statement: executes code if condition is True
- elif: checks another condition if the previous was False
- else: executes code if all conditions are False
- Syntax: if condition: / elif condition: / else:
- While loop: repeats while condition is True. Use break to exit, continue to skip.
- For loop: iterates over a sequence (list, range, string, etc.)
- range(): generates a sequence. range(5) = 0,1,2,3,4. range(1,6) = 1,2,3,4,5.
- break: exits the loop immediately
- continue: skips the current iteration and goes to next
- pass: does nothing, placeholder


FUNCTIONS:
- Functions are reusable blocks of code defined with def keyword.
- Syntax: def function_name(parameters): / body / return value
- Parameters: variables passed into the function
- Return statement: sends a value back to the caller
- Default parameters: def greet(name="Student"): gives default value if not provided
- *args: accepts variable number of positional arguments
- **kwargs: accepts variable number of keyword arguments
- Local variables: defined inside a function, only accessible inside
- Global variables: defined outside, accessible everywhere. Use global keyword to modify inside function.


LISTS:
- Lists are ordered, mutable, and allow duplicates. Defined with square brackets [].
- Example: fruits = ["apple", "banana", "mango"]
- Indexing: fruits[0] = "apple". Negative: fruits[-1] = "mango"
- Methods: .append(), .remove(), .pop(), .sort(), .reverse(), .len(), .insert(), .clear()
- List slicing: fruits[1:3] returns items at index 1 and 2
- Looping through list: for item in fruits: print(item)
- List comprehension: [x*2 for x in range(5)]


TUPLES:
- Tuples are ordered and immutable (cannot be changed after creation). Defined with parentheses ().
- Example: coordinates = (10, 20)
- Tuples are faster than lists and used for data that should not change.
- Tuple unpacking: x, y = (10, 20)


SETS:
- Sets are unordered, unindexed, and do not allow duplicates. Defined with curly braces {}.
- Example: colors = {"red", "green", "blue"}
- Methods: .add(), .remove(), .union(), .intersection(), .difference()
- Used to remove duplicates from a list: list(set(my_list))


DICTIONARIES:
- Dictionaries store data in key-value pairs. Defined with curly braces {}.
- Example: student = {"name": "Riwaj", "age": 21, "grade": "A"}
- Accessing values: student["name"] or student.get("name")
- Methods: .keys(), .values(), .items(), .update(), .pop()
- Looping: for key, value in student.items(): print(key, value)


EXCEPTION_HANDLING:
- try block: code that might cause an error
- except block: handles the error if it occurs
- else block: runs if no error occurred
- finally block: always runs regardless of error
- Common exceptions: ValueError, TypeError, ZeroDivisionError, FileNotFoundError, KeyError, IndexError, NameError
- Raising exceptions: raise ValueError("Custom message")
- Syntax:
  try:
      risky_code()
  except ValueError as e:
      print(f"Error: {e}")
  finally:
      print("Done")


OOP_CLASSES_AND_OBJECTS:
- A class is a blueprint for creating objects.
- Syntax: class ClassName:
- __init__: constructor method, runs when object is created. def __init__(self, params):
- self: refers to the current object instance
- Creating an object: obj = ClassName(arguments)
- Instance attributes: stored in self. Example: self.name = name
- Methods: functions inside a class
- Encapsulation: bundling data and methods together in a class


INHERITANCE:
- Inheritance allows a child class to inherit attributes and methods from a parent class.
- Syntax: class ChildClass(ParentClass):
- super(): calls the parent class constructor. super().__init__()
- Method overriding: child class can redefine a parent method
- isinstance(): checks if object is instance of a class
- Types: single inheritance (one parent), multiple inheritance (multiple parents)


SCOPE:
- Local scope: variables defined inside a function, only accessible there
- Global scope: variables defined outside all functions, accessible everywhere
- global keyword: allows modifying a global variable inside a function
- NameError occurs when trying to access a variable outside its scope


MODULES:
- A module is a Python file containing functions, classes, and variables.
- Importing: import module_name or from module_name import function_name
- Built-in modules: math, random, datetime, os, sys
- math module: math.sqrt(), math.pi, math.ceil(), math.floor()
- random module: random.random(), random.randint(), random.choice(), random.seed()
- datetime module: datetime.datetime.now(), datetime.timedelta(days=5)
- Creating custom modules: save a .py file and import it by filename


INTROSPECTION:
- Code introspection means examining objects to know their type, attributes, and methods at runtime.
- type(): returns the type of an object
- dir(): returns list of attributes and methods
- help(): returns documentation
- isinstance(): checks if object is of a given type
- hasattr(): checks if object has an attribute


CLOUD_COMPUTING_AND_PYTHON:
- Python is widely used in cloud computing for automation, scripting, and building cloud applications.
- Task-specific Python libraries allow building specialized applications quickly.
- Python's simplicity makes it ideal for cloud-based development.
- Examples of cloud use cases: web APIs, serverless functions, data pipelines, AI models.
